# LPX批量重命名工具

一个基于PyQt6的现代化批量文件重命名工具，专为批量处理文件名而设计。

## 功能特点

- 🎯 **智能项目匹配**: 基于项目代号自动匹配和生成新文件名
- ⚙️ **灵活差分规则**: 支持自定义差分规则配置，包括版本名、缩写和语言
- 📁 **拖拽支持**: 支持直接拖拽文件或文件夹到窗口中
- 🔍 **查找替换**: 内置查找替换功能，批量修改文件名
- 💾 **配置管理**: 支持保存和切换多套配置
- 📜 **操作历史**: 完整的操作记录和撤销功能
- 🌐 **记忆库功能**: 智能记忆常用的版本名、缩写和语言
- 🎨 **现代界面**: 基于现代化设计的用户界面

## 文件说明

- `美化_renamer.py` - 主启动器，负责检查更新并启动主程序
- `app_logic.py` - 核心逻辑代码，包含主要的重命名功能
- `LPX批量重命名工具.exe` - 打包好的可执行文件
- `configs/` - 配置文件存储目录
- `auto_config.json` - 自动保存的配置文件
- `memory_bank.json` - 记忆库数据文件
- `window_config.json` - 窗口配置文件

## 使用方法

### 方法一：直接运行exe文件
双击 `LPX批量重命名工具.exe` 即可运行

### 方法二：Python源码运行
```bash
# 确保已安装Python 3.7+和所需依赖
pip install PyQt6 requests

# 运行程序
python 美化_renamer.py
```
