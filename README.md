# LPX批量重命名工具

一个基于PyQt6的现代化批量文件重命名工具，专为批量处理文件名而设计。
完全由ai生成，自用的小工具，不适用所有人。

## 功能特点

- 🎯 **智能项目匹配**: 基于项目代号自动匹配和生成新文件名
- ⚙️ **灵活差分规则**: 支持自定义差分规则配置，包括版本名、缩写和语言
- 📁 **拖拽支持**: 支持直接拖拽文件或文件夹到窗口中
- 🔍 **查找替换**: 内置查找替换功能，批量修改文件名
- 💾 **配置管理**: 支持保存和切换多套配置
- 📜 **操作历史**: 完整的操作记录和撤销功能
- 🌐 **记忆库功能**: 智能记忆常用的版本名、缩写和语言
- 🎨 **现代界面**: 基于现代化设计的用户界面
